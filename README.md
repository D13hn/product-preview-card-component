# Product Preview Card Component

This is a solution to the "Product Preview Card" challenge from **Frontend Mentor**.

## 🚀 1. Overview
* This project was built to strengthen my skills in **CSS Flexbox** and **Responsive Web Design**.
* The focus was on creating a fully responsive layout that switches from a mobile-first approach to a desktop-optimized view.

## 🛠 2. Technologies Used
1. **HTML5**
2. **CSS3** (using Media Queries for responsiveness).
3. **Mobile-first workflow**.

## 💡 3. What I Learned
1. **Deepened Concepts:** This exercise was a turning point for me. By rebuilding the project from scratch, I moved beyond just "following tutorials" to truly understanding how browser rendering works and how CSS properties interact.
2. **Layout Management:** I mastered the art of using `display: none` and `display: block` to manage asset visibility across different breakpoints effectively.
3. **Flexbox Mastery:** I gained a solid understanding of how to use Flexbox to create balanced, centered, and flexible layouts that adapt to any screen size.
4. **Problem-Solving:** My debugging skills were significantly sharpened. I learned that the best way to fix a complex responsiveness issue is not to patch it, but to step back, analyze the root cause, and rebuild the structure with a clearer logic.

## 📱 4. Challenges Encountered & Solutions
1. **The Responsive Hurdle:** Initially, I struggled with the responsiveness while transitioning between mobile and desktop views, as the images wouldn't align correctly.
2. **The Solution:** I took the bold step of wiping the code and restarting. This process helped me identify where I was overcomplicating the logic. By simplifying my CSS structure and strictly adhering to mobile-first principles, I successfully resolved the layout issues and ensured a seamless transition across all devices.

## 🚀 5. Final Thoughts
This challenge has been instrumental in solidifying my confidence as a web developer. It wasn't just about finishing the task; it was about learning how to think like a developer.
